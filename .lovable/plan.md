
# Магазин тюльпанов «Москва» — план

Создадим лёгкий весенний интернет-магазин с акварельной эстетикой, отойдя от шаблонного e-com через journal/editorial-сетку, мягкие пастельные тона и рукописные акценты.

## Визуальный язык
- **Палитра:** кремовый фон #FBF7F2, пудровый розовый, мятный, лавандовый, акценты тёплого терракота
- **Типографика:** изящный serif (Cormorant / Playfair) для заголовков + чистый sans (Inter) для текста; рукописные подписи к секциям
- **Детали:** мягкие тени, скруглённые формы, акварельные пятна-блобы как фоны секций, плавные hover-анимации, ассиметричные сетки вместо ровных карточек

## Структура сайта (публичная часть)

**Хедер** — лого «Тюльпаны», навигация (Каталог, Доставка, О нас, Контакты), иконка корзины со счётчиком, телефон Москвы

**Главная (/)**
- Hero: крупное фото-композиция с тюльпанами, рукописный заголовок «Весна приходит букетами», CTA «Смотреть каталог»
- Блок «Сегодня в наличии» — 3-4 акцентных букета в editorial-сетке
- «Тюльпаны поштучно» — выбор от 25/51/101 шт с превью
- Секция «Как мы работаем» — 3 шага (выбор → заявка → доставка) с акварельными иллюстрациями
- Отзывы / instagram-полоса
- CTA-блок доставки по Москве

**Каталог (/catalog)**
- Фильтры в боковой панели: Тип (букеты / поштучно), Цвет (розовые, белые, жёлтые, микс, пионовидные), Количество, Цена
- Сортировка: по цене, новинки
- Карточки в editorial-стиле — фото крупно, название курсивом, цена, кнопка «В корзину»

**Карточка товара (/catalog/$id)**
- Большая фотогалерея слева, информация справа
- Название, описание, состав, цена
- Селектор количества стеблей (если поштучно) или вариаций
- «В корзину» + «Купить в 1 клик»
- Блок «Уход за тюльпанами», «Доставка и оплата»
- Похожие букеты

**Корзина (/cart)**
- Список товаров с превью, изменение количества, удаление
- Итого, поле промокода (визуально)
- Форма оформления: Имя, Телефон, Адрес доставки, Дата/время, Комментарий
- Кнопка «Оставить заявку» → создаёт заказ в БД, показывает страницу благодарности

**Доставка (/delivery)** — зоны по Москве, время, стоимость, самовывоз

**О нас (/about)** — короткая история, ценности, фото

**Контакты (/contacts)** — телефон, адрес, мессенджеры, карта

**Политика конфиденциальности (/privacy)** и **Публичная оферта (/oferta)** — отдельные страницы с типовым текстом

**Футер** — навигация, контакты, ссылки на правовые документы, соцсети, копирайт

## Админка (/admin)

**Вход (/admin/login)** — email + пароль через Lovable Cloud Auth, доступ только пользователям с ролью `admin` (отдельная таблица user_roles, без хранения роли в profiles)

**Дашборд (/admin)** — счётчики: новых заявок, всего товаров, заявок за неделю

**Заявки (/admin/orders)**
- Таблица: дата, имя, телефон, сумма, статус (новая / в работе / выполнена / отменена)
- Открытие заявки в модалке: состав заказа, контакты, изменение статуса, заметки

**Товары (/admin/products)**
- Список с поиском и фильтром по типу
- Кнопка «Добавить товар» → форма: название, описание, цена, тип (букет/поштучно), цвет, количество стеблей, фото (загрузка в Cloud Storage), наличие, активность
- Редактирование/удаление любого товара

## Бэкенд (Lovable Cloud)

**Таблицы:**
- `products` — товары (id, name, description, price, type, color, stems_count, images[], in_stock, active)
- `orders` — заявки (id, customer_name, phone, address, delivery_date, comment, total, status, created_at)
- `order_items` — позиции заказа (order_id, product_id, quantity, price)
- `user_roles` — роли пользователей (user_id, role enum: admin/user) — отдельная таблица для безопасности
- Storage bucket `product-images` для фото

**Безопасность:** RLS на всех таблицах. Публично читаются только активные товары. Заявки может создавать кто угодно, читать/менять — только админы (через security-definer функцию `has_role`). Товары изменяют только админы.

## Что получите
Полнофункциональный сайт-витрину с акварельным журнальным дизайном, корзиной и оформлением заявок без оплаты, плюс защищённую админку для приёма заявок и управления каталогом.
